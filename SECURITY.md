## Reporting a Vulnerability

If you discover a security vulnerability in **PhenoEvents**, please report it responsibly:

1. **DO NOT** create a public GitHub issue for security vulnerabilities.
2. Send a private report via GitHub's [Private vulnerability reporting](https://github.com/KooshaPari/PhenoEvents/security/advisories/new)

## Security Best Practices

When using this library:

- Keep dependencies up to date
- Review code that uses `unsafe`
- Validate all inputs
- Never log sensitive information
